1. <html>
    • Description: Defines the root of an HTML document.
    • Usage: It encapsulates the entire HTML document, including all other elements.
  2. <head>
    • Description: Contains meta-information about the HTML document.
    • Usage: It typically includes elements like <title>, <meta>, <link>, <script>, etc., which provide information about the document rather than content.
  3. <title>
    • Description: Sets the title of the HTML document.
    • Usage: The text inside this tag appears as the title in the browser's title bar or tab.
  4. <body>
    • Description: Contains the visible content of the HTML document.
    • Usage: It includes all the elements that are meant to be displayed to the user, such as text, images, links, etc.
  5. <h1> to <h6>
    • Description: Define headings of different levels.
    • Usage: <h1> represents the most important heading, while <h6> represents the least important. They are used to structure the content hierarchically.
  6. <p>
    • Description: Defines a paragraph.
    • Usage: It is used to group a block of text content together within the document.
  7. <a>
    • Description: Defines a hyperlink
    • Usage: It creates a link to another web page, file, email address, or any other URL.
  8. <img>
    • Description: Embeds an image into the HTML document.
    • Usage: It displays an image on the web page, with the 'src' attribute specifying the image file's URL.
  9. <ul>
    • Description: Defines an unordered list.
    • Usage: It creates a bulleted list of items where the order does not matter.
  10. <ol>
    • Description: Defines an ordered list.
    • Usage: It creates a numbered list of items where the order does matter.
  11. <li>
    • Description: Defines a list item
    • Usage: It is used within '<ul>' or '<ol>' to define each individual item in the list.
  12. <div>
    • Description: Defines a division or a section in an HTML document.
    • Usage: It is used as a container for other HTML elements, allowing you to group them together and apply styles or manipulate them as a whole.
  13. <span>
    • Description: Defines a section in an HTML document for styling purposes.
    • Usage: It is used to apply styles to a specific part of the text or to group inline elements together for styling purposes.
  14. <input>
    • Description: Defines an input control.
    • Usage: It creates an interactive control element, such as a text field, checkbox, radio button, etc., allowing users to input data.
  15. <button>
    • Description: Defines a clickable button.
    • Usage: It creates a clickable button that can trigger JavaScript functions or submit form data.
